## 📌 Project Overview
This is a Machine Learning web application that predicts house prices based on key features like income, house age, number of rooms, bedrooms, and population.

The model is trained using **Random Forest Regressor** and deployed using **Flask**.

---

## 🧠 Machine Learning Workflow
- Data Collection (housing.csv)
- Data Preprocessing
- Feature Selection
- Model Training (Random Forest)
- Model Evaluation (MAE, RMSE)
- Model Deployment using Flask
